# Eğitim Platformu

Bu proje, basit bir eğitim platformu web sitesi örneğidir. HTML ve CSS kullanılarak oluşturulmuştur. Responsive tasarımla mobil ve masaüstü cihazlarda uyumlu çalışır.

## Özellikler

- **Responsive Tasarım**: Mobil, tablet ve masaüstü için optimize edilmiş.
- **Kurs Listesi**: 6 farklı kurs sunuluyor (Web Geliştirme, Python, Veri Bilimi vb.).
- **Sosyal Medya Bağlantıları**: GitHub, LinkedIn ve Twitter linkleri.
- **Modern UI**: Temiz ve kullanıcı dostu arayüz.

## Kullanılan Teknolojiler

- **HTML5**: Semantik yapı için.
- **CSS3**: Stil ve responsive tasarım için (Flexbox, Media Queries).
- **Resimler**: Kurs görselleri ve sosyal medya ikonları.

## Kurulum ve Çalıştırma

1. Bu repository'yi klonlayın:

   ```
   git clone https://github.com/Omer1640/egitim-platformu.git
   ```

2. Proje klasörüne gidin:

   ```
   cd egitim-platformu
   ```

3. `index.html` dosyasını bir web tarayıcısında açın (örneğin, Chrome, Firefox).

Başka bir kurulum gerektirmez – tamamen statik bir sitedir.
